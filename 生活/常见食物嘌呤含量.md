## 嘌呤含量分类

| 分类名称  | 嘌呤含量 | 食物举例                                                       |
| --------- | -------- | -------------------------------------------------------------- |
| 🔴高嘌呤   | >150     | 肝、肾；海苔、紫菜(干)；鲭鱼、贻贝、生蚝、海兔、鱿鱼等         |
| 🟠较高嘌呤 | 75~150   | 牛肉、猪肉、羊肉；兔、鸭、鹅；鲤鱼、比目鱼、草鱼等             |
| 🟡较低嘌呤 | 30~75    | 大米、燕麦、荞麦；豆角、菜花；香菇(鲜)、金针菇(鲜)、口蘑(鲜)等 |
| 🟢低嘌呤   | <30      | 马铃薯、甘薯；胡萝卜、油菜、生菜、竹笋；水果类；奶及奶制品等   |
